## CTG Scraper

##### Web scraper for [Cleaning The Glass](https://www.cleaningtheglass.com)

##### Technologies Used
- Materialize
- Handlebars
- Node
- MongoDB
- Mongoose
- Cheerio

### Start
Clone the repo here [CTG Scraper](https://github.com/tastaub/mongooseScraper)
Run:
> npm i
Or visit at
> [CTG Scraper](https://stark-crag-37953.herokuapp.com/)
